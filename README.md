# BienAca Control

Server application for the Bien Aca application.

## Installation

Note: _Make sure you have installed rvm_.

```bash
$> git clone git@github.com:ingenious-agency/bien-aca-backend.git
$> cd bien-aca-backend
$> bundle install
```

## Current features

- [x] Register a user
- [x] Login as a user
- [x] Send a heartbeat
- [x] Admin interface
